# Too Much Left

这是一个可以通过 .gitignore 过滤规则来匹配文件进行文件删除的工具

## 特性

### 搜索功能

- 选择执行目录进行文件搜索
- 支持将文件按大小进行过滤
- 支持指定文件大小比较方法，小于或大于等于指定大小的比较
- 支持自定义文件大小进行比较
- 支持搜索规则定义，可定义**包含列表**和**排除列表**，搜索将匹配**包含列表**中的文件或目录并排除**排除列表**中的文件或目录
- 支持保存并管理搜索规则
- 支持退出时当前规则与保存规则的对比，提示用户进行规则保存
- 支持多线程并发搜索
- 支持取消搜索功能

### 文件列表

- 支持批量非阻塞内容渲染
- 支持批量文件选择
- 支持上下文菜单: 打开文件目录、移除文件、复制路径
- 支持字段排序
- 支持多线程并发删除
- 支持删除进度条
